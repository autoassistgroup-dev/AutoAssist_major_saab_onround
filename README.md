# AutoAssist Portal

A comprehensive support ticket management system for AutoAssistGroup.

## Features
- **Ticket Management**: Create, track, and manage support tickets.
- **Admin Panel**: Manage members, technicians, and system settings.
- **Technician Management**: Track technician status and assignments.
- **Dashboard**: Real-time overview of ticket status and metrics.
- **Member Portal**: User access for creating and viewing tickets.

## Tech Stack
- **Backend**: Python (Flask)
- **Database**: MongoDB
- **Frontend**: HTML, TailwindCSS, JavaScript
- **Authentication**: Custom session management

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/autoassistgroup-dev/AutoAssist__portal.git
   ```

2. Create a virtual environment:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

4. Configure Environment:
   Create a `.env` file based on your configuration requirements.

5. Run the application:
   ```bash
   python app.py
   ```
